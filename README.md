## ✨ Features
- Auto-collapse all GitHub Actions job logs on page load
- Manually expanded steps stay open (smart state)
- Toggle on/off via toolbar icon
- Lightweight, no tracking, no external requests

## 🔒 Privacy Policy
**This extension does not collect, store, or transmit any user data.**

| Permission | Purpose |
|------------|---------|
| \`storage\` | Save enabled/disabled state locally |
| \`host_permissions\` | Modify UI only on \`github.com/*/actions/runs/*\` |

- ❌ No analytics • ❌ No external calls • ✅ All logic runs locally
